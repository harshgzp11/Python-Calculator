# Python Calculators (Basic + Scientific)

This repository contains two Python calculator programs created for learning and practice.  
Both calculators are completely menu-driven and beginner-friendly.

### 1. Basic Calculator
A simple calculator that supports:
- Addition
- Subtraction
- Multiplication
- Division

This version is built using only:
- Loops  
- Conditionals  
- Basic input/output  

Perfect for beginners learning Python fundamentals.

---

### 2. Scientific / Advanced Calculator
A complete menu-driven scientific calculator with the following features:

#### Basic Operations
- Addition  
- Subtraction  
- Multiplication  
- Division  

#### Advanced Operations
- Modulo  
- Exponentiation  
- Floor Division  

#### Scientific Operations
- Square Root (using `math.sqrt`)  
- Cube Root (power-based)  

#### Extra Features
- Calculation History (stores all previous calculations)  
- Clear History option  
- Exit option  
- Implemented without functions or exception handling (beginner version)
